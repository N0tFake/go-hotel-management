# Hotel management

Hotel room management API

## Dependencies
- Golang
- Gorm
- Gim
- Viper
- PostgreSQL

## Use
Remove ```.dist``` from ```app.env.dist``` file to use environment variables.

install [air](https://github.com/cosmtrek/air) to live reload
```
go install github.com/cosmtrek/air@latest
```

Execute:
```
air
```
